# 当当网图书数据爬取与可视化系统

这是一个基于Flask的Web应用，用于爬取当当网图书销量数据并进行可视化分析。

## 功能特点

- 🕷️ 爬取当当网畅销书排行榜数据
- 📊 多维度数据可视化（饼图、柱状图）
- 📈 实时数据统计分析
- 🎨 现代化的Web界面

## 技术栈

- **后端**: Flask
- **爬虫**: requests + parsel
- **数据处理**: pandas
- **可视化**: pyecharts
- **前端**: HTML + CSS + JavaScript

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行应用

```bash
python app.py
```

应用将在 http://localhost:5000 启动

## 使用说明

1. 打开浏览器访问 http://localhost:5000
2. 设置要爬取的页数（默认5页）
3. 点击"开始爬取数据"按钮
4. 等待爬取完成后，点击"生成可视化图表"
5. 查看数据统计和可视化图表

## 可视化内容

- **图书原价区间分布**: 饼图展示不同价格区间的图书数量分布
- **图书售价TOP15**: 柱状图展示售价最高的15本书
- **图书评论数TOP15**: 柱状图展示评论数最多的15本书，包含平均值、最大值、最小值标记线

## 数据字段

爬取的数据包含以下字段：
- 书名
- 评论数
- 推荐量
- 作者
- 出版社
- 售价
- 原价
- 折扣
- 电子书价格
- 详情链接

## 注意事项

- 爬取数据时请遵守网站的robots.txt协议
- 建议设置合理的爬取间隔，避免对服务器造成压力
- 数据仅供学习和研究使用
